# Minnesota Boot Camp Module 5 Challenge

## Description

Module Five challenge for the Minnesota Boot Camp. A work day scheduler that saves events per hourly time block, with the text elements being saved across browser refreshes by pulling from localstorage. Includes changing elements depending on the users hour on their local machine.
